# Challenge-Alura-Store
Este proyecto analiza el desempeño de cuatro tiendas de la cadena Alura Store utilizando técnicas de análisis de datos con Python.


El propósito del análisis realizado.
  El objetivo de este proyecto es analizar el desempeño de cuatro tiendas de la cadena Alura Store utilizando técnicas de análisis de datos en Python.

  El análisis busca identificar cuál de las tiendas presenta el menor rendimiento para recomendar cuál debería vender el Sr. Juan al iniciar un nuevo emprendimiento.

  Para ello se analizaron diferentes indicadores clave:
  
  ingresos totales de cada tienda
  ventas por categoría de productos
  calificación promedio de los clientes
  productos más y menos vendidos
  costo promedio de envío

  A partir de estos análisis se generaron visualizaciones que permiten comprender el comportamiento de las ventas y el desempeño general de cada tienda.

Ejemplos de gráficos e insights obtenidos.
  Durante el análisis se generaron diferentes visualizaciones para comprender mejor los datos.

  Facturación por tienda
  Este gráfico permite comparar los ingresos totales de cada tienda.
  Insight principal:
      Tienda 1 presenta la mayor facturación.
      Tienda 4 presenta la menor facturación.

  Ventas por categoría
  Este análisis muestra qué categorías de productos tienen mayor demanda.
  Insights:
    Las categorías Muebles y Electrónicos concentran la mayor cantidad de ventas.

  Calificación promedio de clientes
  Este gráfico muestra el nivel de satisfacción de los clientes por tienda.
  Insights:
    Tienda 3 tiene la mejor calificación promedio.
    Las diferencias entre tiendas son relativamente pequeñas.

  Costo promedio de envío
  Este análisis permite evaluar la eficiencia logística.
  Insights:
    Tienda 1 presenta el costo de envío más alto.
    Tienda 4 presenta el costo de envío más bajo.
  
  Conclusión del análisis
  Después de evaluar todos los indicadores —facturación, categorías de productos, calificaciones, rotación de productos y costos de envío— se concluye que:
  
  La Tienda 4 presenta el menor rendimiento general, principalmente debido a su menor nivel de facturación.
  Por esta razón, se recomienda considerar la venta de la Tienda 4, concentrando recursos en las tiendas con mejor desempeño.
